SIDVI Sistema de información y diagnóstico de virus

¿Qué es?

Una plataforma de acceso fácil y rápido donde los usuarios podrán consultar un catálogo de virus. Cada uno de ellos incluye 4 secciones.
Información para consultar datos o información general.
Test donde puedes realizar un pre-diagnóstico.
Estadísticas donde el usuario puede ver visualmente el progreso del virus. Se pueden crear gráficas nuevas para visualizar datos específicos filtrar por una variedad de campos como localidad, genero, edad, sexo, entre otros.
Médicos Disponibles donde se puede consultar el catálogo de consultorios que atienden dicho virus.
Este sistema es escalable ya que en cualquier momento se puede agregar un nuevo virus al catálogo junto con su información. La lista de medicos que atienden cada virus es auto gestionable ya que son los mismo medicos quienes dan de alta su consulsultorio para poder aparecer en SIDVI, despues de una validación por parte del sistema.

Misión

Otorgar información valiosa, eficaz y veridica para prevenir posibles contagios, realizar pre-diagnósticos e informar a la sociedad a través de una plataforma de acceso rápido.

Visión

Ser la plataforma número uno para consultar información y promocionar consultorios acerca de virus en distintas localidades del mundo.
